# 系统性能优化知识体系

## 核心性能指标
- CPU使用率/负载
- 内存占用/缺页率
- 磁盘IOPS/吞吐
- 网络带宽/延迟
- 应用级QPS/时延

## 工具矩阵
```mermaid
flowchart TB
    监控工具-->|系统级|perf
    监控工具-->|内核级|ftrace
    监控工具-->|应用级|eBPF
    分析工具-->火焰图
    分析工具-->CPU缓存分析
    压测工具-->wrk2
    压测工具-->sysbench
```

## 优化方法论
1. **问题定位**：建立监控基线，使用`perf top`快速定位热点
2. **根因分析**：结合火焰图和硬件事件采样
3. **方案实施**：代码优化/算法改进/架构调整
4. **效果验证**：AB测试对比性能指标
